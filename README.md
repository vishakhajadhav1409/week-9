# 🚀 Task Management RESTful API (Flask + JWT)

A complete RESTful API for a task management system built using Flask. This project demonstrates backend development, API design, authentication, and database management.

---

## 📌 Features

- 🔐 JWT Authentication (Login/Register)
- 👤 User Management
- ✅ Task CRUD Operations
- 🔍 Filtering, Sorting, Pagination
- ⚡ Rate Limiting
- 🛡️ Input Validation & Error Handling
- 📄 Swagger API Documentation
- 🧪 Unit Testing
- 📦 Postman Collection

---

## 🛠️ Tech Stack

- Python 3.x
- Flask
- Flask-RESTful
- Flask-JWT-Extended
- Flask-SQLAlchemy
- Flask-Migrate
- Flask-Limiter
- SQLite / PostgreSQL
